‘AntiProcess:Bonding’ is an audio-visual performance that is part of a game currently under development. Centring on a fictional virtual world, it aims to create a model that abstracts reality. It is a schizophrenic simulation of an ambiguous and chaotic place that adapts to the player’s actions. The performance presents the ‘initial state’ of the work where the forms are pure, still somewhat orderly, on the cusp of something more entropic.

Specs:

- Realtime audiovisual composition

- Built in the Unity engine

- Procedurally generated composition

- Shader based GPU rendered graphics

Footage from exhibition at the Talbot Rice Gallery, May-June 2018
